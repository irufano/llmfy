# Configuration
## OpenAI models
To use `OpenAIModel`, add below config to your env:
- `OPENAI_API_KEY`

## AWS Bedrock models
To use `BedrockModel`, add below config to your env:
- `AWS_ACCESS_KEY_ID` 
- `AWS_SECRET_ACCESS_KEY` 
- `AWS_BEDROCK_REGION`

## Example

`.env` file:
```shell
DEBUG=true
VERSION="0.0.0"
OPENAI_API_KEY=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_BEDROCK_REGION=
```
